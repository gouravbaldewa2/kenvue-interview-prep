# Kenvue Innovation Head — 5-Day Interview Prep Programme

Interactive interview preparation guide for the **Innovation Head, Hydration Portfolio** role at Kenvue India.

## What's inside

- **5-Day structured prep plan** with expandable day tiles
- **15+ interview questions** with toggle-able sample answer directions
- **8 STAR stories** mapped to specific resume bullets and JD requirements
- **Competitive landscape** analysis (Electral, Cipla, HUL, Reliance, etc.)
- **Interactive checklist** with progress tracking (saved to browser localStorage)
- **Key numbers** reference section

## How to use

1. Open `index.html` in any browser
2. Click day tiles to expand each day's prep content
3. Click questions in the Q&A Bank to reveal answer directions
4. Use filters to focus on Innovation / Behavioural / Motivation / Killer questions
5. Track progress with the interactive checklist

## Tech

- Pure HTML, CSS, JavaScript — no dependencies, no build step
- Dark theme with responsive design
- Scroll-reveal animations
- LocalStorage for checklist persistence

- **Role:** Innovation Head, Hydration Portfolio — Kenvue India (Director-level)
- **Key brands:** ORSL, eRZL, WHO ORS
- **Candidate profile:** Marketing Manager at Coca-Cola India (Sprite), 11+ years across FMCG and startups
